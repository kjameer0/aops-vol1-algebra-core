# Fluency v1 — Absolute Value Equations and Inequalities

1. Solve for $x$: $\;|2x - 3| = 7$.

2. Solve for $x$: $\;|x + 4| \le 6$. Write the solution as an interval.

3. Solve for $x$: $\;|3x - 1| \ge 8$. Write the solution as a union of intervals.

4. Write a single absolute value inequality in $x$ whose solution set is the open interval $(-2,\, 8)$.

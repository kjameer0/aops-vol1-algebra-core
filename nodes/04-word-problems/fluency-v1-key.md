# Fluency Worksheet v1 — Answer Key

1. Let x = Maria's age, y = son's age.
   - Equations: `x = 4y`, `x + 12 = 2(y + 12)`
   - Answer: Maria is **24**, son is **6**.

2. Let t = hours until they meet.
   - Equation: `50t + 70t = 300`
   - Answer: **t = 5/2 hours (2.5 hours)**.

3. Let x = liters of 20% solution, y = liters of 50% solution.
   - Equations: `x + y = 12`, `0.2x + 0.5y = 0.3(12)`
   - Answer: **x = 8 L of 20%, y = 4 L of 50%**.

4. Let n = number of nickels, d = number of dimes.
   - Equations: `n + d = 25`, `5n + 10d = 185` (cents)
   - Answer: **n = 13 nickels, d = 12 dimes**.

5. Let x and y be the two numbers, x > y.
   - Equations: `x + y = 42`, `x - y = 8`
   - Answer: **x = 25, y = 17**.

6. Total time = downstream time + upstream time.
   - Expression: `30/(24 + 6) + 30/(24 - 6)`
   - Answer: **8/3 hours (≈ 2.667 hours, or 2 hours 40 minutes)**.

7. Let t = Tom's age, j = Jerry's age.
   - Equations: `t = j + 3`, `t + j = 27`
   - Answer: **Tom = 15, Jerry = 12**.

8. Let a = adult tickets, c = child tickets.
   - Equations: `a + c = 200`, `8a + 5c = 1375`
   - Answer: **a = 125 adult tickets, c = 75 child tickets**.
